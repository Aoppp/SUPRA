# SUPRA 网页部件命名参考

以后直接说"部件名 + 要改什么"即可，如："把搜索结果表格的驱动力标签列改成紫色"。

## 页面一：主搜索页 (`/`) — SearchPage.tsx

```
┌──────────────────────────────────────────┐
│  页面标题                                  │
├──────────────────────────────────────────┤
│  搜索面板（白色圆角卡片）                     │
│  ┌──────┐ ┌──────┐ ┌──────┐              │
│  │搜索栏1│ │搜索栏2│ │搜索栏3│              │
│  ├──────┤ ├──────┤ ├──────┤              │
│  │搜索栏4│ │搜索栏5│ │搜索栏6│              │
│  ├──────┤ ├──────┤ ├──────┤              │
│  │搜索栏7│ │搜索栏8│ │      │              │
│  └──────┘ └──────┘ └──────┘              │
│                                          │
│  [🔍 搜索按钮]                              │
├──────────────────────────────────────────┤
│  结果摘要行："找到 10 条结果"                 │
├──────────────────────────────────────────┤
│  搜索结果表格（白色卡片，可横向滚动）            │
│  ┌────────┬──────┬──────┬──────┬──────┐  │
│  │ 化合物  │ 图片  │CAS号 │组装  │驱动  │..│  ← 表头行
│  ├────────┼──────┼──────┼──────┼──────┤  │
│  │Gallic  │ 🖼️   │149.. │Self..│H-b.. │  │  ← 结果行
│  └────────┴──────┴──────┴──────┴──────┘  │
├──────────────────────────────────────────┤
│  分页器：[1] [2] [3]                       │
└──────────────────────────────────────────┘
```

### 搜索面板（SearchPanel）

| 编号 | 部件名 | 类型 | 例子 |
|------|--------|------|------|
| 搜索栏1 | **组装体名称输入框** | `<input>` 文本 | 占位符 "e.g., Fmoc-FF hydrogel" |
| 搜索栏2 | **构建基元下拉框** | `<select>` | 选项如 "Gallic Acid", "Quercetin"... |
| 搜索栏3 | **形貌下拉框** | `<select>` | 选项如 "Nanofiber", "Hydrogel"... |
| 搜索栏4 | **驱动力下拉框** | `<select>` | 选项如 "Hydrogen bonding", "π-π stacking"... |
| 搜索栏5 | **性质下拉框** | `<select>` | 选项如 "Antibacterial", "Drug delivery"... |
| 搜索栏6 | **溶剂输入框** | `<input>` 文本 | 占位符 "e.g., Water" |
| 搜索栏7 | **最小尺寸输入框** | `<input>` 数字 | 占位符 "e.g., 10" |
| 搜索栏8 | **最大尺寸输入框** | `<input>` 数字 | 占位符 "e.g., 200" |
| — | **搜索按钮** | `<button>` 蓝色 | 显示 "搜索" 或 "搜索中..." |

### 搜索结果表格（ResultsTable）— 8 列

| 列名 | 部件名 | 内容样式 | 例子 |
|------|--------|----------|------|
| 化合物 | **化合物名称列** | 蓝色链接，点击跳转详情页 | "Gallic Acid" |
| 图片 | **化合物图片列** | 缩略图或 "—" | PubChem PNG 或灰色占位 |
| CAS号 | **CAS号列** | 等宽小字 | "149-91-7" |
| 组装类型 | **组装类型列** | 小字，超出截断 | "Self-assembly; Hydrogel" |
| 驱动力 | **驱动力标签列** | 琥珀色小标签 | `[Hydrogen bonding]` |
| 形貌 | **形貌列** | 普通文字 | "Rod-like nanofiber" |
| 粒径 | **粒径列** | 小字，不换行 | "~1 μm (fiber diameter)" |
| 生物活性 | **生物活性标签列** | 绿色小标签 | `[Antibacterial]` |

### 其他

| 部件名 | 描述 |
|--------|------|
| **结果摘要行** | "找到 10 条结果 · 第 1 / 5 页" |
| **分页器** | 底部页码按钮组 |
| **空结果提示** | "未找到结果。" |

## 页面二：详情页 (`/assembly/:id`) — DetailPage.tsx

```
┌──────────────────────────────────────────┐
│  ← 返回搜索                                 │
├──────────────────────────────────────────┤
│  组装体标题（大号加粗）                        │
│  形貌副标题（灰色小字）                        │
├──────────────────────────────────────────┤
│  详情卡片网格 (2列布局)                       │
│  ┌────────────────┐ ┌────────────────┐    │
│  │ 基本信息卡片     │ │ 描述卡片        │    │
│  └────────────────┘ └────────────────┘    │
│  ┌────────────────────────────────────┐    │
│  │ 制备方法卡片（跨两列）                  │    │
│  └────────────────────────────────────┘    │
│  ┌────────────────┐ ┌────────────────┐    │
│  │ 构建基元卡片     │ │ 表征方法卡片     │    │
│  └────────────────┘ └────────────────┘    │
│  ┌────────────────┐ ┌────────────────┐    │
│  │ 驱动力卡片       │ │ 性质卡片（标签云）│    │
│  └────────────────┘ └────────────────┘    │
└──────────────────────────────────────────┘
```

### 详情卡片

| 部件名 | 位置 | 内容样式 |
|--------|------|----------|
| **返回链接** | 顶部 | 蓝色文字 "← 返回搜索" |
| **组装体标题** | 顶部 | 大号加粗 |
| **形貌副标题** | 标题下方 | 灰色小字 |
| **基本信息卡片** | 左上 | 溶剂/浓度/尺寸范围/DOI 键值对 |
| **描述卡片** | 右上 | 段落文字 |
| **制备方法卡片** | 中（全宽）| 编号步骤文字 |
| **构建基元卡片** | 左下 | 名称/分子式/类别/SMILES 键值对 |
| **表征方法卡片** | 右下 | 名称/类别 键值对 |
| **驱动力卡片** | 左下 | 无序列表，附类别 |
| **性质与应用卡片** | 下（全宽）| 蓝色标签云 |

## 页面三：浏览页 (`/browse`) — BrowsePage.tsx

与搜索页结果表格结构相同，但没有搜索面板，直接展示全部数据。
列：编号 / 名称 / 构建基元 / 形貌 / 尺寸。

## 全局部件 — Layout.tsx

| 部件名 | 描述 |
|--------|------|
| **网站标题** | 左上角 "SUPRA" |
| **搜索导航链接** | 导航栏 "搜索" / "Search" |
| **浏览导航链接** | 导航栏 "浏览" / "Browse" |
| **语言切换按钮** | 右上角 "中文" 或 "EN" |

## 文件对照

| 文件 | 对应的页面/部件 |
|------|---------------|
| `frontend/src/components/Layout.tsx` | 顶部导航栏、网站标题、导航链接、语言切换按钮 |
| `frontend/src/components/LanguageToggle.tsx` | 语言切换按钮 |
| `frontend/src/pages/SearchPage.tsx` | 主搜索页（搜索面板 + 搜索结果表格 + 分页器） |
| `frontend/src/pages/DetailPage.tsx` | 详情页（所有详情卡片） |
| `frontend/src/pages/BrowsePage.tsx` | 浏览页 |
| `frontend/src/i18n/translations.ts` | 所有文本的中英文翻译 |
| `frontend/src/api/client.ts` | 前端 API 请求函数 |
| `frontend/src/types/index.ts` | TypeScript 类型定义 |
| `backend/models.py` | 数据库表结构 |
| `backend/schemas.py` | API 请求/响应数据结构 |
| `backend/crud.py` | 数据库查询逻辑 |
| `backend/seed.py` | 种子数据（虚构示例数据） |
| `backend/main.py` | API 路由入口 |
